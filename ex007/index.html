Texto marcado

Podemos também criar um texto marcado usando a tag < MARK >

Texto grande e pequeno

Podemos cria texto com letras Grande com a tag < BIG >
uma tag que esta saido do html

Também podemos fazer textos com letras Pequana com a tag ≶ SMALL >
uma tag que continua no html

Teste de paradigima

teste para colocar as tag big e small juntas

texto com big dentro do small.
texto com small detro de big.

Em ambos os caso as letras continua com tamanho normal

Texto deletado

podemos marca um texto como excluido com a tag < DEL > (< DEL >)
para indicar que ele deve ser lindo, mas não considerado

texto inserido

Podemos marca um texto como inserido para da enfase e indicar que ele foi adicionado depois.
Pra isso usando a tag < ins > , uma tag com semantica.

também podemos marca um texto com inserido usando a tag < U > .
essa tag não tem semantica.

Texto sobrescrito

Texto sobrescrito são expoente, para isso é usando a tag < sup >
exemplo: 3x 5+7x 9+10x 11
os numeros sobrescrito são: 5,9 e 11.

texto subscrito

Texto subscrito é bem utilizado em quimica, para isso é usando a tag < sub >
exemplo: H 2 O
os numero subscrito é somente: 2
