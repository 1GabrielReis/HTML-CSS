A acessar a link externos, é necessario usa a tag < A >

A tag para link externos é < a >

Tag para muda de pagina com no mesmo arquivo:
< A href="pag2.html" rel="netx" > < /A >
Tag para muda de pagina em um arquivo interno:
< A href="interno/pag3.html" rel="netx" > < /A >
interno/: nome do arquivo dentro do arquivo principal.
pag3.html: nome do site.
